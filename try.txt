# Remove current remote
  git remote remove origin

  # Add new remote
  git remote add origin YOUR_NEW_REPO_URL

  # Verify the change
  git remote -v

  Or to update the existing remote URL:

  # Change the URL directly
  git remote set-url origin YOUR_NEW_REPO_URL

  # Verify
  git remote -v

  Example with a new GitHub repo:
  git remote set-url origin git@github.com:YOUR_USERNAME/nqr-landingpage.git

  Or with HTTPS:
  git remote set-url origin https://github.com/YOUR_USERNAME/nqr-landingpage.git
